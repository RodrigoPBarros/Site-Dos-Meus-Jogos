# 🎮 GAME ARCHIVE v3.0 // CYAN PROTOCOL

Sistema de gerenciamento de backlog e biblioteca de jogos pessoal, otimizado para setups de alta performance com estética industrial.

---

## 🖥️ THE SETUP
Este software foi projetado para rodar em um ambiente específico:
* **Monitor:** 24 polegadas.
* **Aesthetic:** Black & White com detalhes em Cyan Neon.
* **Furniture:** Estilo industrial (Ferro e Madeira).
* **Peripheral:** Layout totalmente wireless (Mouse, Teclado e Controles) com cable management avançado.

## ⚙️ TECNOLOGIAS & ENGINE
* **Interface:** Glassmorphism Industrial (Deep Charcoal & Cyan).
* **Core:** Vanilla JavaScript com persistência via `localStorage`.
* **Rendering:** Módulos de categoria dinâmicos e filtros de plataforma (PC/EMU).

## 🕹️ DATABASE SYNCED
O arquivo conta com uma biblioteca diversificada, incluindo:
* **TOP 7 Essentials:** Títulos como *The Witcher 3* e *God of War (2018)*.
* **Soulslike & Metroidvania:** Coleção completa de *Dark Souls* e clássicos como *Hollow Knight*.
* **Retro Archives:** Títulos de PS2 e emuladores rodando em modo "Smooth".

---

## 🛠️ COMO INSTALAR
1. Mantenha os arquivos `index.html`, `style.css` e `script.js` na mesma pasta.
2. Certifique-se de que os cabos do seu setup estão devidamente roteados atrás dos móveis para manter a imersão.
3. Abra o `index.html` no seu navegador de preferência (recomenda-se modo tela cheia F11).
